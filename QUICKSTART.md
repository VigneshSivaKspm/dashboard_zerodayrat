# Quick Start Guide - MDM Dashboard

## 🚀 Getting Started

### 1. Access the Application

Navigate to the login page:
```
/login
```

**Demo Credentials:**
- Email: `admin@mdm.com`
- Password: Any password (demo mode)

### 2. Navigation

After login, you'll see the main dashboard with a sidebar containing:

- 📊 **Overview** - Global fleet status
- 📱 **Device Fleet** - All devices list
- 💾 **Data Logs** - Notifications, SMS, Activity
- 📂 **Media Manager** - Photos and files
- 📈 **Analytics** - Charts and reports
- ⚙️ **Settings** - Configuration

### 3. Key Actions

#### View Device Details
1. Go to **Device Fleet**
2. Click on any device row
3. Explore Overview, Remote Control, and History tabs

#### Send Remote Commands
1. Navigate to a device detail page
2. Click **Remote Control** tab
3. Use command buttons:
   - 📱 Screen Share
   - 📸 Capture Photo
   - 🎤 Record Audio
   - 📳 Vibrate
   - 👁️ Toggle Visibility

#### Monitor Activity
1. Go to **Data Logs**
2. Switch between tabs:
   - Notifications
   - SMS (with send capability)
   - Timeline

#### View Media
1. Go to **Media Manager**
2. Browse:
   - Gallery (photos/videos)
   - Files (documents)

#### Export Data
1. Go to **Analytics**
2. Scroll to Export Hub
3. Select data types
4. Choose format (CSV/JSON/Excel)
5. Click **Export Data**

### 4. Search & Filter

**Global Search:**
- Use the search bar in the header
- Search by device ID or label

**Page Filters:**
- Device Fleet: Filter by status
- Data Logs: Search notifications/SMS
- Media Manager: Filter by device/date

### 5. Real-time Features

Watch for these indicators:
- 🟢 **Live dot** - Real-time syncing
- 🔴 **Offline badge** - Device disconnected
- 🟡 **Low battery** - <20% charge
- ⚡ **Activity feed** - Updates every 5 seconds

### 6. Settings Configuration

1. Click profile icon (top right)
2. Go to **Settings**
3. Configure:
   - Account info
   - Security & 2FA
   - Firebase connection
   - Notifications
   - Data retention

### 7. Mobile Responsive

The dashboard is fully responsive:
- Mobile: Collapsible sidebar
- Tablet: Optimized layouts
- Desktop: Full feature set

## 🎯 Common Tasks

### Add New Device
1. Device Fleet → **Add New Device**
2. Enter device details
3. Save

### Send SMS from Device
1. Data Logs → SMS tab
2. Click **Send SMS**
3. Select device
4. Enter number and message
5. Send

### View Command History
1. Device Detail → **History** tab
2. See all executed commands
3. Check success/failure status

### Check Fleet Health
1. Overview page
2. Review stat cards
3. Check offline devices
4. Monitor battery levels

## 💡 Tips

- **Hover Effects**: Hover over cards and rows for actions
- **Status Colors**: Green=online, Red=offline, Amber=warning
- **Quick Access**: Click device names anywhere to view details
- **Bulk Actions**: Use checkboxes for multiple devices
- **Export Regularly**: Use Analytics to export important data

## 🔧 Troubleshooting

**Device Not Showing:**
- Check network connection
- Verify Firebase sync status
- Check device registration

**Commands Not Executing:**
- Ensure device is online
- Check command history for errors
- Verify permissions

**Data Not Loading:**
- Check "All Systems Live" indicator
- Refresh the page
- Check Firebase configuration
